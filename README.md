# DDRigMonitoring

DDRigMonitoring

DDRigMonitoring เป็นเว็บแอปพลิเคชันที่ใช้สำหรับ ตรวจสอบสถานะของโปรดักที่อยู่ใน RIG Station

เว็บนี้ทำหน้าที่ อ่านข้อมูล Status จากฐานข้อมูล (Database) เท่านั้น
การเปลี่ยนแปลง Status จะเกิดขึ้นจาก แอปพลิเคชัน Delphi (DDRigTester.exe)
แต่ละสถานะของโปรดักจะมีการแสดงผลด้วยสีต่าง ๆ เพื่อให้ง่ายต่อการอ่าน:
เทา (Gray) – ว่าง / Empty
ส้ม (Orange) – กำลังทดสอบ / Testing
น้ำเงิน (Blue) – กำลังตรวจสอบ / Checking
เขียว (Green) – ผ่านการทดสอบ / Passed
แดง (Red) – ล้มเหลว / Failed
ดำ (Black) – ถูกล็อก / Locked
เว็บจะมีการแสดงผลเป็นตาราง พร้อม Timer และ Status ของแต่ละโปรดัก เพื่อให้สามารถติดตามเวลาที่ใช้และสถานะล่าสุดของโปรดักแต่ละตัวได้อย่างชัดเจน
